clinic-management-system-java

A simple Java-based Clinic Management System for managing patient data using CRUD operations, searching, sorting algorithms, and file handling (CSV).

---

Features
- Add new patient data (Create)
- Display all patient records (Read)
- Edit patient information (Update)
- Soft delete patient (Deactivate)
- Search patient by name (Linear Search)
- Search patient by ID (Binary Search)
- Search patient by category (Linear Search)
- Sort patients by ID (Bubble Sort)
- Sort patients by name (Selection Sort)
- Sort patients by visit count (Descending)
- Basic statistics (total and active patients)
- Data storage using CSV file

---

Tech Stack:
- Java (OOP)
- File Handling (CSV)
- Basic Data Structures & Algorithms
